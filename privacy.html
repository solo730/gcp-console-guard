GCP Console Guard Privacy Policy

Last updated: April 2026

Overview

GCP Console Guard is a read-only Chrome extension designed to help users navigate the Google Cloud Console quickly by providing one-click access to any GCP service.

Data Collection

GCP Console Guard does not collect, transmit, or sell personal data. All functionality is performed locally within your browser.

  • No Google Cloud credentials are accessed or stored
  • No data is sent to external servers
  • No tracking, analytics, or advertising is used

Local Storage

The extension uses chrome.storage.local to store:

  • Your Pro license key (if purchased), stored only on your device
  • License activation status

This data never leaves your device and can be cleared at any time using the Remove license button inside the extension.

GCP Console Access

GCP Console Guard operates only on pages under https://console.cloud.google.com and relies on your existing Google Cloud session. It reads the current page DOM to detect your project ID for display purposes only. It does not modify resources or make API calls to Google Cloud.

License Validation

If you purchase a Pro license, your license key is sent to our secure validation server solely to verify its validity. No personal information is transmitted beyond the license key itself.

Third Parties

GCP Console Guard does not integrate with third-party analytics or advertising services. License validation is handled through LemonSqueezy and a Cloudflare Worker proxy.

Disclaimer

GCP Console Guard provides informational navigation only and is not affiliated with Google or Google Cloud.
